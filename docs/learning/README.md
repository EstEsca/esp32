# 학습 회고

세션을 진행하면서 **배운 것**을 기록하는 자리. 코드는 세션 폴더에, 회고는 여기에.

권장 형식:

```
NN-session-name.md          ← 세션 단위 회고
topic-<주제>.md              ← 부품·기법 단위 깊이 있는 정리
```

예시 후보:
- `00-bootstrap.md` — PIO 환경 잡으면서 헤맨 것
- `topic-tft_espi-setup.md` — TFT_eSPI build_flags 정리
- `topic-encoder-debounce.md` — 인터럽트 기반 엔코더 처리

> 비교: 사용자용 매뉴얼은 [../manual/](../manual/), 부품·핀 스펙은 [../../hardware/](../../hardware/)에.
